¿Qué es una Raspberry Pi?

El cerebro de nuestro proyecto: tecnología de alta eficiencia y bajo costo.

Raspberry Pi 3 Model B

Computadora monoplaca de bajo costo, consumo energético mínimo (≈ 5W) y alta versatilidad.

Costo Extremadamente Bajo

Permite equipar escuelas rurales con presupuestos mínimos en comparación con PCs de escritorio convencionales.

Bajo Consumo de Energía

Puede operar con paneles solares pequeños o baterías portátiles en zonas con suministro eléctrico inestable.

Entorno Offline

El sistema está cargado con todo el software educativo y bases de datos locales, eliminando la necesidad de conectarse a Internet.

La Brecha Digital en Escuelas Rurales

En las comunidades marginadas de nuestro estado, la falta de acceso a recursos tecnológicos y didácticos limita drásticamente el desarrollo de los estudiantes. Escuelas como la Primaria Rural de San Pedro Huamelula, Oaxaca, sufren un aislamiento severo: durante años se reportó que sus agencias carecían de internet, televisión abierta y señal telefónica celular.

Nuestro proyecto combate esta problemática de raíz: llevando hardware asequible y software cargado de información a las manos de los alumnos, sin depender de redes de telecomunicaciones inexistentes.

"La brecha digital en las escuelas rurales representa una problemática grave por la carencia de recursos bibliográficos y de trabajo."

— Diagnóstico Tecnologías Educativas, 2026

Software Educativo

Programas Incorporados

Detalle de las herramientas seleccionadas para promover el aprendizaje autónomo y habilidades digitales.

KIWIX

Servidor de Contenido Offline

Es un servidor de contenido web offline que opera en segundo plano. No necesita una interfaz gráfica, los usuarios acceden remotamente conectándose a la IP de la Raspberry desde cualquier navegador web ligero.

50 MB 40 - 60 MB RAM
  • Acceso a Wikipedia y libros locales.
  • Operación sin conexión a Internet.
  • Acceso multidispositivo mediante red Wi-Fi local.

GCompris-Qt

Suite de Aprendizaje Lúdico

Suite interactiva de actividades educativas locales. Funciona como una aplicación que interactúa directamente con el servidor gráfico del sistema operativo, proveyendo juegos de matemáticas, lectura y ciencias.

400 MB 200 MB RAM
  • Más de 100 actividades educativas.
  • Apto para alumnos de nivel primaria.
  • Interfaz intuitiva y adaptada a niños.

AbiWord

Procesador de Textos Ligero

Procesador de textos modular que actúa como un binario ligero, carga solo las bibliotecas esenciales de manipulación de texto bajo una interfaz GTK eficiente y amigable.

150 MB 60 MB RAM
  • Redacción de reportes y tareas escolares.
  • Interfaz clásica de herramientas familiares.
  • Muy bajo consumo de recursos (ideal para RPi).

GNumeric

Hoja de Cálculo Independiente

Hoja de cálculo que funciona como una herramienta independiente y ligera. Ejecuta algoritmos de cálculo matemático directamente en la memoria local, facilitando el análisis y la estadística básica.

Incluido en GTK 40 MB RAM
  • Operaciones aritméticas y matemáticas locales.
  • Ideal para la iniciación en cómputo administrativo.
  • Carga instantánea sin latencia en Raspberry.

Manual del Usuario y Flasheo

Instrucciones sencillas para preparar tu memoria SD y poner a funcionar la computadora.

1

Descargar Imagen

Descarga la imagen comprimida del sistema pre-cargado desde nuestro repositorio oficial de recursos digitales.

2

Flashear Memoria SD

Inserta una memoria MicroSD de 32GB en tu computadora local y usa la herramienta Raspberry Pi Imager o BalenaEtcher para grabar la imagen descargada en ella.

Inserta la tarjeta SD en la ranura de la Raspberry Pi, conecta un teclado, un mouse, una pantalla o televisión por HDMI y el cable de alimentación. ¡El sistema cargará automáticamente listón para aprender!

Referencia Rápida

Comandos Básicos de la Terminal

Guía esencial de comandos para navegar y operar tu Raspberry Pi desde la línea de comandos.

Navegación de Archivos

ls

Lista los archivos y carpetas del directorio actual.

ls -la

Lista todos los archivos (incluyendo ocultos) con detalles.

cd nombre_carpeta

Entra a una carpeta específica.

cd ..

Regresa a la carpeta anterior (un nivel arriba).

cd ~

Regresa a la carpeta personal del usuario (Home).

pwd

Muestra la ruta completa del directorio actual.

Gestión de Archivos

mkdir nueva_carpeta

Crea una nueva carpeta.

touch archivo.txt

Crea un archivo vacío nuevo.

cp archivo.txt copia.txt

Copia un archivo con un nuevo nombre.

mv archivo.txt carpeta/

Mueve o renombra un archivo.

rm archivo.txt

Elimina un archivo (¡cuidado, no va a la papelera!).

rm -r carpeta/

Elimina una carpeta y todo su contenido.

Sistema y Procesos

sudo apt update

Actualiza la lista de paquetes disponibles.

sudo apt upgrade

Instala las actualizaciones de todos los paquetes.

sudo apt install nombre

Instala un programa nuevo desde los repositorios.

sudo reboot

Reinicia la Raspberry Pi.

sudo shutdown -h now

Apaga la Raspberry Pi de forma segura.

htop

Muestra los procesos activos y el uso de CPU/RAM en tiempo real.

Información y Red

hostname -I

Muestra la dirección IP de tu Raspberry Pi.

df -h

Muestra el espacio disponible en disco (memoria SD).

free -h

Muestra el uso de memoria RAM.

cat /proc/cpuinfo

Muestra información del procesador.

uname -a

Muestra información completa del sistema operativo.

vcgencmd measure_temp

Muestra la temperatura actual del CPU.

Consejo: La tecla TAB es tu mejor amiga

Mientras escribes un comando o nombre de archivo, presiona TAB para autocompletar. Presiona TAB dos veces para ver todas las opciones disponibles. Esto te ahorrará mucho tiempo y evitará errores de escritura.

De Windows a Raspberry Pi

Preguntas Frecuentes

Respuestas a las dudas más comunes para quienes vienen de Windows y usan por primera vez un sistema basado en Linux.

Sí, el escritorio usa un entorno llamado LXDE/LXQt que es más ligero que Windows. Pero la idea es la misma: tienes una barra de tareas abajo, un menú de inicio arriba a la izquierda, y puedes abrir aplicaciones haciendo clic. Los iconos del escritorio funcionan igual que en Windows: doble clic para abrir.

En lugar de descargar un .exe de Internet como en Windows, en Raspberry Pi se usa la terminal con el comando sudo apt install nombre_del_programa. Es más seguro porque todos los programas vienen de repositorios verificados. También puedes usar el menú "Add/Remove Software" que funciona como una tienda visual similar a la Microsoft Store.

No existen letras de unidad como C: o D:. En Linux todo parte de una sola raíz /. Tu carpeta personal está en /home/tu_usuario/ y es el equivalente a C:\Users\TuNombre\ en Windows. Dentro encontrarás carpetas como Escritorio, Documentos y Descargas, igual que en Windows.

sudo significa "Super User DO" y es equivalente a "Ejecutar como Administrador" en Windows. Lo usas cuando necesitas permisos elevados para instalar software o cambiar configuraciones del sistema. En cuanto al antivirus: Linux es mucho más seguro por diseño. No necesitas antivirus, ya que el sistema de permisos y los repositorios verificados protegen tu equipo de forma nativa.

No es obligatorio. Puedes usar la Raspberry Pi de forma completamente gráfica: abrir programas desde el menú, mover archivos con el explorador de archivos, y navegar con el mouse como en Windows. La terminal es una herramienta poderosa pero opcional para tareas avanzadas. Con el tiempo, descubrirás que muchas cosas son más rápidas desde la terminal.

Si necesitas compatibilidad completa, también puedes instalar LibreOffice, que es el equivalente completo y gratuito de Microsoft Office en Linux.

¡Claro que sí! Solo conecta tu memoria USB y aparecerá automáticamente en el escritorio o en el explorador de archivos, similar a Windows. Para expulsarla de forma segura, haz clic derecho en el icono de la USB y selecciona "Expulsar" o "Desmontar". La Raspberry Pi 3 tiene 4 puertos USB disponibles.

¡Nunca desconectes el cable directamente! Esto puede dañar la tarjeta SD y perder tus archivos. Para apagarla correctamente, ve al menú de inicio y selecciona "Shutdown", o abre la terminal y escribe sudo shutdown -h now. Espera a que la luz verde del LED deje de parpadear antes de desconectar el cable de alimentación.
